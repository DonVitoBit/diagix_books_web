# Руководство пользователя

## Вход в систему

1. Откройте приложение в браузере (например, http://localhost:8501).
2. Введите логин и пароль (по умолчанию: `admin` / `admin123`).
3. Нажмите «Войти».

Роли:
- **Администратор** — полный доступ ко всем разделам.
- **Модератор** — только вкладка «Книги» с доступными книгами.

---

## Вкладка «Книги»

- Просмотр списка переписанных книг.
- Открытие книги для просмотра и редактирования.
- Сохранение изменений, создание версий.
- Комментарии к абзацам.
- Восстановление предыдущих версий.
- (Админ) Управление доступом модераторов к книгам.

---

## Вкладка «Перефразирование»

### Перефразирование по документам

1. Загрузите файл (PDF, TXT, MD, DOCX).
2. Укажите тему текста.
3. Выберите уровень перефразирования (temperature).
4. Нажмите «Перефразировать».

Текст будет обработан блоками с сохранением научной терминологии.

### Генерация статьи по теме (без плана)

1. Введите тему.
2. (Опционально) Укажите поисковые запросы для PubMed.
3. Нажмите «Сгенерировать статью».

Статья генерируется в реальном времени (стриминг).

### Генерация статьи по плану (Gemini/OpenAI)

1. Введите тему.
2. Нажмите «Сгенерировать план» — появится структура статьи с разделами.
3. (Опционально) Включите «Добавить иллюстрации» — в текст будут вставлены маркеры и сгенерированы изображения.
4. Нажмите «Сгенерировать статью».

Система:
- Строит план с searchQueries для PubMed.
- Ищет релевантные статьи в PubMed.
- Генерирует текст по плану с учётом источников.
- При включённых иллюстрациях — генерирует промпты, создаёт изображения (NanoBanana или DALL-E) и вставляет их в текст.

---

## Вкладка «Результаты»

- Сравнение оригинального и перефразированного текста.
- Статистика (количество слов, изменение объёма).
- Скачивание файлов.

---

## Вкладка «Иллюстрации»

- Извлечение изображений из PDF.
- Классификация (энциклопедические / клинические).
- Поиск замен (TCIA, Google Custom Search).
- Генерация новых изображений (NanoBanana, DALL-E 2).
- Перерисовка выбранных изображений с настройкой промпта.

---

## Вкладка «Настройки»

### Провайдер LLM

Выберите: **OpenAI**, **DeepSeek** или **Gemini**.

### API-ключи

- **OpenAI** — для перефразирования и DALL-E 2.
- **DeepSeek** — альтернатива OpenAI.
- **Gemini** — Google Gemini API.
- **NanoBanana** — генерация изображений.
- **DALL-E 2** — генерация/перерисовка изображений.
- **Google Custom Search** — поиск изображений.

### Параметры

- **Temperature** — уровень творчества при перефразировании (0.0–1.0).
- **Макс. токенов** — лимит для ответов модели.
- **Лимит токенов для статей** — для генерации статей.
- **Добавлять исследования** — вставка ремарок из PubMed в перефразированный текст.

### Управление стилем статьи (1–5)

- Научность, глубина, точность, читаемость, качество источников.

## Вкладка «Модераторы» (только админ)

- Список модераторов.
- Добавление модератора (имя, логин, пароль).
- Смена пароля.
- Удаление модератора.

## Temperature — рекомендации

| Тип текста | Temperature | Описание |
|------------|-------------|----------|
| Медицинские документы | 0.1–0.2 | Минимальные изменения, точность терминов |
| Научные статьи | 0.3–0.5 | Умеренное перефразирование |
| Учебные материалы | 0.4–0.6 | Баланс ясности и вариативности |
| Популярная наука | 0.6–0.8 | Более свободная формулировка |

## Частые вопросы

**Текст не перефразируется**  
Проверьте API-ключ и провайдер в Настройках. Убедитесь, что есть квота.

**Статья генерируется пустой**  
Попробуйте другую тему или провайдер. Проверьте лимит токенов для статей.

**Иллюстрации не вставляются**  
Нужен ключ NanoBanana или DALL-E. При ошибке NanoBanana используется fallback на DALL-E.

**WebSocket connection failed**  
При доступе через nginx — см. раздел «Развёртывание» в документации.
