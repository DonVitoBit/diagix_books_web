## Краткое описание проекта

**Text Rephraser** — приложение для интеллектуальной обработки научных текстов:

- **Перефразирование** — переформулировка с сохранением терминологии (OpenAI, DeepSeek, Gemini)
- **Генерация статей** — создание статей по теме с планом и PubMed-источниками
- **Иллюстрации** — извлечение, генерация (NanoBanana, DALL-E 2), вставка в текст
- **Книги** — хранение переписанных книг, версии, комментарии, доступ модераторов
- **Экспорт** — PDF, LaTeX

## Быстрый старт

```bash
# Установка
python3 -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt

# Запуск веб-приложения
python run_web_app.py

# Или CLI
python main.py --input-file input/book.pdf --theme "МЕДИЦИНА"
```

Вход: `admin` / `admin123` (изменить в `config.py`).
